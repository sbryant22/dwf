Four Websites

Smashing Magazine

This website contains information and links about web site design, different coding languages, how-to tutorials on programming, inspiration, Adobe software products, etc. It is a very helpful site.

It did not pass validation in either of the validation sites. This site has 12 errors but 24 in total, if you count the two together.

To fix these errors, they would need to close their empty tags and to add whitespace between a closing quote.


Logo Lounge

This website is where designers can go for inspiration for their own logos and it also acts like a blog. They have design news, updates, and new works from other designers.

This site did not pass the validation check from the two validation websites. On the HTML Lint site, it stated that it could not continue with the check because the source code has an illegal closing tag. On the validator.nu site, it has 50 errors.

To fix some of these errors I would fix the illegal closing tag by getting rid of the second paragraph end tag (</p>). For a lot of the other errors, I would put in the required attributes and put in the correct values for shapes like, polygon and rectangles.


Universal Orlando

This website is for Universal Studios in Orlando, Florida. One of the popular theme parks in Florida.

This site did not pass the validation check. On the HTML Lint site, it could not continue with the check because an illegal closing tag was used. On the validator.nu site, Universal Orlando has 129 errors in total.

I would fix some of these errors by correcting the illegal closing tag in the head tag. The meta tag is closed incorrectly within the head tag. There are also a lot of missing attributes. I would go back and include the correct ones like, </a>, in most cases. A space between attributes is also required. Quotes also need to match but in this code they don't, so that would need to be addressed as well.


Hot Topic

This website is for a popular Punk-Rocking/music-lover store based in malls across the country. They are all about the music and pop culture with their rocker style clothing, accessories, and their wide selection of CDs.

This site did not pass validation. On HTML Lint, it also had the illegal use of a closing tag with one of their script tags. On the validator.nu site, Hot Topic had a total of 27 errors when you don't count the warnings.

For most of the errors, required attributes are either missing or not allowed. Other errors are that there are misplaced non-spaced characters. To fix most of these errors, the correct attributes will need to be put in. With other errors, like the element link is not allowed in the element center in this context, I would remove that element out of the center element.
