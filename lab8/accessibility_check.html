Smashing Magazine

smashingmagazine.com

Accessibility Review

Smashing Magazine is a very useful website for designers, web designers, and web programmers alike. It has useful tips on all the programming languages and on all of the Adobe software like Photoshop and Illustrator (to just name a few). They also have books and ebooks on mobile and web design.

AChecker.ca and CynthiaSays.com

This site did not pass the WCAG 2.0 or the Section 508 accessibility verification.

This site has 36 errors for WCAG 2.0. Some errors are because valid alt text for the images are missing, appropriate labels are missing, and errors with their CSS. You can see the full list at achecker.ca/checker/index.php

This site also has errors with their Section 508. The errors for this page are harder to figure out in some areas but for the most part, alt, title, and label attributes are not being used when needed. You can see the full list at cynthiasays.com/Home.aspx?

To fix these errors, I would fix the contrasting color errors with the CSS, add label elements that surround the <select> elements, and then add text to describe the function or purpose of the controls in the <input> elements.

Rolling Stone Magazine

rollingstone.com

Accessibility Reveiw

Rolling Stone Magazine's website is where you can go to read up on the latest music, movie, television, and celebrity news.

AChecker.ca and Cynthiasays.com

This site did not pass the WCAG 2.0 and Section 508 accessibility validation.

This site has 43 WCAG 2.0 errors. Some of the errors are that a few <img> elements are missing alt attributes and <input> elements are either missing associated labels or they have the labels but no text is in them. Anchors don't have any text in them. Incorrect headers were used and document language was not identified.

This site has errors with their Section 508 validation. For the most part, the errors are associated with the <img> elements. The appropriate alt attributes/titles are missing. Frame elements don't have the appropriate element content and some of the <input> elements don't have alt, title, or label attributes.

To fix these errors, I would put in descriptive alt attributes for the <img> elements and I would give the <input> elements label attributes and include text in them. Text would also be put into the anchors and the headers would be put in correctly. I would also identify the document language at the beginning of the html code.

Haagen Dazs

haagendazs.com

Accessibility Review

The Haagen Dazs' website features all of their classic and limited edition ice cream flavors. You can learn more about the company's history, you can connect with them through Facebook and Twitter, and by signing up for their newsletter. You can also find a shop near you and look around the site to find your perfect flavor and figure how how you would want it - whether in a waffle cone or in a bowl. This site can make you very hungry.

AChecker.ca and Cynthiasays.com

This site does not pass the WCAG 2.0 and the Section 508 accessibility validation.

This site has 26 WCAG 2.0 errors. Like a lot of the other sites, this one is also missing valid alt text within their <img> elements. Labels are missing. The <script> elements are not keyboard accessible and event handlers are missing for <onmouseover>. The document language wasn't identified and label text are empty.

Haagen Dazs has a few errors with Section 508. The IFRAME elements are missing content and labels are missing for the <input> elements.

To fix these errors, I would put in descriptive alt attributes text into the <img> elements. Labels and their text would be placed into the <input> elements. The script will become keyboard accessible by putting in the appropriate <onmouseout> and <onblur> elements and I would put in the correct document language at the beginning of the HTML.

Dodge Challenger

dodge.com

Accessibility Review

On Dodge's website it features all of their cars and I chose the Dodge Challenger page because I love this car. It gives you information on the car, from the price to how much gas mileage, and it lets you customize the vehicle anyway you would like it. The site also provides a game called Dodge Social letting you explore the vehicle that is featured in the new SCI FI show Defiance.

AChecker.ca and Cynthiasays.com

This site does not pass the WCAG 2.0 and the Section 508 validation.

This site has 22 WCAG 2.0 errors. Alt attributes are missing in the <img> tags. Headers that are following the <h1> tags are incorrect, document language is not identified, and an <id> is not unique.

This site has a lot of errors with Section 508 because the <img> elements do not have alt text. To pass Section 508 all IMG elements require this.

To fix these errors, I would put in the required alt attributes and text into the IMG elements, I would identify the document's language at the beginning of the HTML, and I would correct the headers that are following the <h1> elements.
