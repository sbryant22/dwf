Element descriptions

h1
A level one heading element, used to define the most important heading.
<h1>This is a level one heading example.<h1>
h2
A level two heading element, used to define the sub-heading.
<h2>This is a level two heading example.<h2>
h3
A level three heading element, used to define the Sub sub-heading.
<h3>This is a level three heading element.<h3>
p
A paragraph element, used to define a line (or lines) of text.
<p>This is a paragraph element.<p>
ul
An unordered (bulleted) list element, used for bulleted items
  • <ul>This is an unordered list element.<ul>
ol
An ordered (numbered) list element, used for numbering items in a list.
  1. <ol>This is an ordered list element.<ol>
li
An list item element, used with the <ul>, <ol> elements for when making lists.
  • <li>This is a list item element.<li>
